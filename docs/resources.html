2. RESOURCESÂ¶

The resources subfolder consists of US census shapefile data, taken from this URL. The exploded ZIP archive of US counties was downloaded on 15 APRIL 2020 from cb_2018_us_county_500k.zip. The FIPS data processed from the SHP files is stored in resources/fips_2018_data.pkl.gz.

I also want to find recent data that organizes counties (by FIPS code) into metropolitan areas. I was able to find the Excel spreadsheet of the 2019 metropolitan statistical areas. I have downloaded this Excel spreadsheet in resources/msa_2019.csv, and the Excel spreadsheet version resources/msa_2019.xlsx.

Finally, the file resources/README_template.rst is a template file that covid19_update_readme uses to regenerate the top-level README.rst.
