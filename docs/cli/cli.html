3. USING THE CLIÂ¶

There are six command line interfaces that have been defined here.

The first CLI, covid19_update_database, only updates the NY Times COVID-19 repository with the latest data.

The next two CLIs, covid19_create_movie_or_summary and covid19_state_summary, synthesize the COVID-19 movies and figures, that represent cumulative cases and deaths, for metropolitan statistical areas (MSAs), the CONUS, or individual US states and territories.

The final three CLIs, covid19_movie_updates, covid19_post2server, and covid19_update_readme, are designed for external services to deploy summary data on specific US regions into other locations.
